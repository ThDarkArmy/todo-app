## Description

Todo app with docker and authentication.

## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Running the app in docker container

```bash
$ docker compose up
```

## Test

```bash
# unit tests
$ npm run test
```
